#Список дел

Чтобы начать использовать приложение: 
1. написать в терминале 'python -m pip install -r requirements.txt' - установать небзодимые библиотеки
2. дальше в терминале прописать 'python manage.py migrate', а после этого 'python manage.py runserver'
3. перейдите в браузер, а дальше на домашнюю страницу приложения (http://localhost:8000/), чтобы присмотреть список задач, который изначально будет пустым

Теперь вы можете начать использовать пользовательский интерфейс для добавления списков дел.
